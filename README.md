## ✨ Características

| Funcionalidad | Descripción |
|---|---|
| 📝 **CRUD Completo** | Crear, editar inline, completar y eliminar tareas |
| 🏷️ **Categorías** | Personal, Trabajo, Estudio, Salud, Otros — con colores |
| 🔴🟡🟢 **Prioridades** | Alta, Media, Baja con indicador visual lateral |
| 📅 **Fechas límite** | Selector de fecha con detección de tareas vencidas |
| 🔍 **Búsqueda** | Filtrado en tiempo real por texto |
| 📊 **Estadísticas** | Dashboard lateral con progreso y desglose por categoría |
| 🌙 **Tema oscuro/claro** | Toggle con persistencia en LocalStorage |
| 💾 **Persistencia** | Todas las tareas se guardan en LocalStorage |
| 🖱️ **Drag & Drop** | Reordena tareas arrastrándolas |
| 📱 **Responsive** | Adaptado a móvil, tablet y desktop |
| 🎨 **Glassmorphism** | Diseño premium con transparencias y degradados |
| ✨ **Animaciones** | Micro-animaciones fluidas para cada acción |
| ⌨️ **Atajos** | Enter para añadir, Escape para cancelar |
| 🔔 **Notificaciones** | Toasts informativos para cada acción |

## 🚀 Demo

Abre directamente el archivo `index.html` en tu navegador — **no necesita servidor ni dependencias**.

## 🛠️ Tecnologías

- **HTML5** — Estructura semántica con accesibilidad
- **CSS3** — Variables CSS, Grid, Flexbox, animaciones @keyframes, glassmorphism
- **JavaScript ES6+** — Clase TaskManager, API Drag & Drop nativa, LocalStorage
- **Google Fonts** — Tipografía Inter

## 📖 Cómo Usar

1. **Clona** el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/mi-gestor-tareas.git
   ```
2. **Abre** `index.html` en tu navegador
3. **¡Listo!** Empieza a gestionar tus tareas

## 📱 Responsive

La aplicación se adapta perfectamente a cualquier dispositivo:
- **Desktop** (>900px) — Layout con sidebar + área principal
- **Tablet** (≤900px) — Estadísticas en scroll horizontal
- **Móvil** (≤600px) — Layout apilado con búsqueda desplegable
- **Smartphones pequeños** (≤380px) — Optimizado con targets táctiles amplios
